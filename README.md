# Demand vs Availability Analysis for Revenue Optimization
## 1. Project Title / Headline

Demand vs Availability Analysis for Revenue Optimization
A data analytics project built using SQL and Power BI to identify supply shortages, quantify revenue loss, and support inventory planning through business KPIs and interactive dashboards.
## 2. Short Description / Purpose

This project analyzes daily demand and product availability to uncover supply gaps and measure their financial impact. Using SQL for data preparation and Power BI with DAX for analytics, the dashboard highlights profit, loss, and shortage trends to enable data-driven inventory decisions.
## 3. Tech Stack

🗄️ MySQL – Data storage and join operations

📊 Power BI Desktop – Dashboard development

🧠 DAX – Measures for profit, loss, averages, and KPIs

📂 Power Query – Data cleaning and transformation
## 4. Data Source

CSV dataset containing daily demand and availability along with product-level information.

The data was imported into MySQL for join operations between the test_environment and product tables, then loaded into Power BI for modeling, DAX calculations, and dashboard creation.
## 5. Features / Highlights
###  •Business Problem

The business faces fluctuating daily demand and limited product availability, resulting in supply shortages and significant revenue loss. Without clear visibility into demand vs supply, it becomes difficult to optimize inventory and maximize profitability.
### • Goal of the Dashboard

To create an analytical dashboard that:

Compares daily demand with availability

Identifies supply shortages

Calculates profit and revenue loss

Tracks average daily demand and availability

Supports better inventory and supply planning
### • Key Metrics (DAX Measures)

Average Demand per Day: 48.65

Average Availability per Day: 24.70

Total Supply Shortage: 61K units

Total Profit: 300.55K

Total Revenue Loss: 7.60M

Average Daily Loss: 2.97K
### • Business Impact & Insights

Average daily availability is significantly lower than demand, leading to consistent supply shortages

A total supply shortage of 61K units resulted in 7.60M in revenue loss

Despite generating 300.55K in profit, unmet demand caused an average daily loss of 2.97K

Improving stock availability to match average demand could substantially reduce losses and increase profitability

The analysis highlights the need for better demand forecasting and inventory optimization during high-demand periods
